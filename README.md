In this Project we aimed to test the security of code generated by chatgpt codex.

The AI was given 3 diffrent seperate promts:

-Test A (control): Functional only prompt only

-Test B: Security-aware prompt

-Test C: Security-template-guided prompt based on owasp top 10

Each test was done in a seperate session instance 

We then tested the output of each test for secuirty vuneralties and compared the results of the 3 tests

In Summary:
While the ability to create fully functing applications in just a few minutes with out the need of any prior coding knolwege and just the idea can be seen as a great even magical tool our research has found that unfortuanley even when security is cosnidered from the start, the code produced by AI LLMs (in our test case codex 5.4) contain far to many vunerlabities to be safely deloyed publicaly to users who may trust these vibe coded apps with senetive data.
